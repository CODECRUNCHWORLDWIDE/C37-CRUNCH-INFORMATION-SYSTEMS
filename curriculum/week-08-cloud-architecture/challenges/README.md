# Week 8 — Challenges

Two challenges. Both stretch past the exercises' guardrails into judgment-call territory — real cloud-architecture work, where the hard part is rarely provisioning a resource and almost always deciding which resources you actually need and which are waste dressed up as safety margin.

## How to work these

- Challenge 1 is a **design** exercise — no required deployment, a written architecture document and diagram.
- Challenge 2 is an **analysis** exercise — you're handed a wasteful (fictional) cloud bill as data and must find and justify cuts using SQL, then verify the savings in Python.
- Neither has one right answer. Both are graded on whether your reasoning is sound and your assumptions are stated, not on matching a key.
