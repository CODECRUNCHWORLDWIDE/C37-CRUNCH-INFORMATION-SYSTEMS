# Week 7 — Challenges

Two challenges. Both stretch past the exercises' guardrails into judgment-call territory — real integration work, where the hard part is rarely the code and almost always the decisions around it.

## How to work these

- Challenge 1 is a **design** exercise — no required code, a written integration design document.
- Challenge 2 is a **code** exercise — you take a working-but-fragile ETL job and make it resilient, with tests proving the resilience.
- Neither has one right answer. Both are graded on whether your reasoning is sound and your assumptions are stated, not on matching a key.
