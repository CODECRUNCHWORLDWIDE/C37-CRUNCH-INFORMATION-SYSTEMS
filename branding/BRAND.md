# C37 · Crunch Information Systems — Brand Guide

- **Accent:** Code Crunch blue `#2463EB` (with deep `#1d4ed8`).
- **Voice:** systems-thinking and pragmatic. Always start from the organization and the problem, then the data, then the technology — never a tool in search of a use. Show the trade-off, name the cost, and design for the people who have to run it. Data lives in SQL and Python, never in a spreadsheet-as-database — and we say why.
- **Icon motif:** connected nodes forming a system — boxes (data + process) linked by arrows (integration), with a spark for the AI layer.
- **One-liner:** "Design the systems a business runs on — data, cloud, automation, and AI, done right."
